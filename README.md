# Luka Dakić Fitness

A professional personal trainer website built for Luka Dakić, based in Bosnia & Herzegovina. Built with Next.js 16, TypeScript, Tailwind CSS, and shadcn/ui — featuring Google Ads conversion tracking, SEO optimization, smooth animations, and a full set of service pages.

## Features

- Multi-page site with dedicated pages for each training service
- Google Ads conversion tracking on contact buttons and floating CTA
- Smooth scroll animations with Framer Motion
- Floating CTA, back-to-top button, and section indicator
- FAQ section
- SEO optimized with next-sitemap, meta tags, and structured data
- Fully responsive — mobile first design
- Dark/light theme support

## Getting Started

1. Clone the repo
   git clone https://github.com/MilosLjubojevic/LukaDakicFitness.git

2. Install dependencies
   npm install

3. Run the development server
   npm run dev

Open http://localhost:3000

## Scripts

npm run dev    - Start development server
npm run build  - Production build
npm run start  - Start production server
npm run lint   - ESLint
